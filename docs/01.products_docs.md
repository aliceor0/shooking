# 상품 목록 페이지 프로젝트

## 프로젝트 개요

- 프로젝트 목표와 주요 기능에 대한 간략한 설명
  1. 프로젝트 목표 : 웹 페이지 기반 모바일 환경의 상품 목록 페이지
  2. 주요 기능
    - 상품 사진 잘 보이게
    - 장바구니 수량 실시간 연동
  
- 참여한 팀원 : korin


## 개발 환경

 React, JavaScript ES6

## 설치 및 실행 방법

```
npm install
npm start
```

## 컴포넌트 구조

주요 컴포넌트들의 구조와 역할

- `ProductList`: 상품 목록을 보여줍니다.
- `ProductCard`: 각 상품의 정보를 보여줍니다.

## 테스트 방법

테스트 케이스 및 실행 방법

```
npm test
```
